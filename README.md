# Portfolio Website

This repository contains the source code for my personal portfolio website, where I showcase my projects, experience, and technical writing.

The site is built to be fast, clean, and easy to iterate on as my work evolves.

## Tech Stack

* React / Next.js
* CSS
* Framer Motion
* MathJax

## Development

```bash
npm install
npm run dev
```
